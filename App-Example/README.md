## Purpose

The purpose of the README.md file is:

- Offer a starting point for people to familiarize themselves with the
  software distribution.
- Describe basic purpose and usage of the software.
- Optionally, give practical intallation instructions (this may also be
  placed in an INSTALL.md file).
- Make note of important points regarding the software that the user
  should be aware of before proceeding with installation
